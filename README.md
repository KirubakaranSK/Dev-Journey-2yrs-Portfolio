## üöÄ Setup & Usage

To run this portfolio locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/KirubakaranSK/Dev-Journey-2yrs-Portfolio.git
    ```
2.  **Navigate to the directory**:
    ```bash
    cd Dev-Journey-2yrs-Portfolio
    ```

    Simply open the `index.html` file in your preferred web browser. No build step is required as it uses vanilla HTML/CSS/JS (with some CDN links).
